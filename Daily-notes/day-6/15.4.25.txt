Types of HTML Elements
1. Block Elements: Element which is occupying parent element size is called block element.

    Eg. headers, paragraph, div, list (ul & ol)

2. Inline Elements: Element which is occupying content size is called inline element.   

    Eg. anchor, text formatting elements, image, span

Float: Using float, we can convert block element into inline element.

    Syntax:

        float: left | right;

        left: It starts displaying the elements from left side.

        right: It starts displaying the elements from right side.
